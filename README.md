# Digital-Kitab-Ghar-Library
ڊجيٽل ڪتاب گھر لائبريري (Digital Kitab Ghar Library) مختصر تعارف: هي هڪ جديد ۽ مڪمل سنڌي ايپليڪيشن آهي، جنهن جو مقصد سنڌي ادب، تاريخ، ۽ شاعريءَ کي ڊجيٽل دور جي گهرجن مطابق پيش ڪرڻ آهي. هي پراجيڪٽ طارق حسين پنهور جي نگرانيءَ ۾ سنڌي پڙهندڙن لاءِ هڪ اهڙو پليٽ فارم مهيا ڪري ٿو، جتي اهي نه صرف ڪتاب پڙهي سگهن ٿا، پر علمي بحث ۽ تحقيق به ڪري سگهن ٿا.
